# 图片去水印工具

本地图片去水印工具，涂抹水印区域即可自动去除，全程在浏览器本地处理，**图片不会上传**到任何服务器。

## 文件说明

| 文件 | 用途 |
|------|------|
| `ImageWR.html` | 电脑版，双击即可用鼠标操作 |
| `mobile.html` | 手机版（PWA），可"添加到主屏幕"离线使用 |
| `manifest.webmanifest` / `sw.js` / `icon-*.png` | PWA 所需文件（离线缓存、图标） |
| `serve.js` | 本地测试服务器 |
| `gen-icons.js` | 重新生成图标（可选） |

## 电脑版

直接双击 `ImageWR.html` 打开即可，无需任何环境。

## 手机版（PWA）

PWA 需要通过 `https://` 或 `localhost` 访问才能安装和离线使用，**直接双击 html 文件（file://）无法注册离线功能**。

### 方式一：局域网测试（推荐先试这个）

电脑装了 Node.js 后，在本目录运行：

```bash
node serve.js
```

终端会打印手机访问地址（形如 `http://192.168.x.x:8080/mobile.html`），手机连**同一个 WiFi**，用浏览器打开该地址即可使用。

> 注意：局域网 `http://` 下大多数浏览器可以正常使用去水印功能，但"安装到主屏幕/离线"因为不是 HTTPS 可能受限。要完整体验安装+离线，用方式二。

### 方式二：免费静态托管（可扫码安装、离线）

把这几个文件一起传到任意支持 HTTPS 的静态托管，即可扫码安装：

- **GitHub Pages**：新建仓库上传全部文件，开启 Pages，访问 `https://<用户名>.github.io/<仓库>/mobile.html`
- **Cloudflare Pages / Vercel / Netlify**：拖拽本文件夹即可部署

需要上传的文件：`mobile.html`、`manifest.webmanifest`、`sw.js`、`icon-192.png`、`icon-512.png`。

打开后：
- **安卓 Chrome**：顶部出现"安装"按钮，或浏览器菜单 → "添加到主屏幕"
- **iPhone Safari**：分享按钮 → "添加到主屏幕"

装好后就有一个独立图标，点开像 App 一样，断网也能用。

## 使用方法

1. 打开/选择带水印的图片
2. 用画笔（手机用手指）把水印涂成红色
3. 点"去除水印"
4. 效果不佳时可反复涂抹、反复去除，或调高"强度"
5. 点"保存"导出结果

手机端：单指涂抹，双指缩放/移动。
